Version Notes

Version 0.7

Notes

  • Contains skater and goalie data from NHL and other leagues (see more details) from 2000-2021
  • Can now handle names with arbitrary capitalization and without diacritics ("Nicklas Backstrom" and "nicklas backstrom" and "Nicklas Bäckström" all return the same result)
  • Updated duration of tournament teams to be more realistic

Known Issues

  • Trade information is incomplete. Specifically, the exact trade dates are missing, so some players may incorrectly appear as teammates if they were traded on the same date between the same teams.
  • Skaters' full names must be match the spelling used by EliteProspects (although diacritics and capitalization are optional). So, for example, "Yevgeni Malkin" will return results but "Evgeni Malkin" will not).
