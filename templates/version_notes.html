Version Notes

Version 0.6

Notes

  • Contains skater and goalie data from NHL and other leagues (see more details) from 2000-2021
  • Can now handle names with arbitrary capitalization and without diacritics ("Nicklas Backstrom" and "nicklas backstrom" and "Nicklas Bäckström" all return the same result)

Known Issues

  • Trade information is incomplete. The exact trade dates are missing, so two players may erroneously appear as teammates or not if they were traded in the same year to/from the same team. Team information for players that were traded multiple times in one year is incomplete.
  • Skaters' full names must be match the spelling used by EliteProspects (although diacritics and capitalization is options). So, for example, "Yevgeni Malkin" will return results but "Evgeni Malkin" will not).
