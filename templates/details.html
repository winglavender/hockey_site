Version 1.1

  • New feature: now you can visualize all the former teammate relationships between two NHL rosters. See some examples here.

Version 1.0

  • New features: now includes (mostly accurate) NHL trade dates.
  • The app focuses on NHL players from 2000-2022. In addition, it contains roster data from the AHL, international tournament teams (Worlds, World Juniors, Olympics), as well as a selection of pro and junior leagues in Canada, the United States, Sweden, Russia, and Finland. See the full list of included leagues).
  • To search for a player, you must use their full name and the exact spelling used by EliteProspects. For example, "Yevgeni Malkin" will return results but "Evgeni Malkin" will not. However, capitalization and diacritics are optional ("Nicklas Backstrom" and "nicklas backstrom" and "Nicklas Bäckström" all return the same result).

The code is available on Github.
